## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Prismic rich text → set:html | HTML rendered via prismic.asHTML() — content comes from Prismic CMS editors, not end users |
| Prismic image URLs → Image component | Prismic CDN URLs allowlisted in image.remotePatterns |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-13 | Tampering | Prismic rich text via set:html | accept | Content authors control Prismic; not end-user input. prismic.asHTML() serializes known node types only. XSS from CMS content is an accepted editorial risk at PoC stage. |
| T-02-14 | Information Disclosure | Hero carousel image alt text | accept | Alt comes from Prismic editorial fields; no secrets in image metadata |
| T-02-15 | Denial of Service | Carousel setInterval never stopped | accept | Interval runs indefinitely while page is open; acceptable for PoC. Phase 5 will add pause-on-interaction. |
</threat_model>
